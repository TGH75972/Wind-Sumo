<b>Wind Sumo is a minecraft minigame just like the classic sumo minigame in which you punch players off the platform, but in wind sumo you can blow em' off the 
platform using the new wind charges added to minecraft! <br>

This minigame can be played by either 2 players or even 3 players if you have some friends to play with.<br>
This server side minigame is made using the Fabric api and it employs the minecraft version 1.21.11 for you to be able to play the game.<br>

The game modes are selected by pressing a button in the lobby you spawn in. If two players press the button, they will be teleported into the arena with unlimited wind <br>
charges but a third player can also join the game within a time limit of 10 seconds when the two players are already in the arena.</b><br>

This means you can either do a 1v1 or a 1v1v1 making it more chaotic with three players or just playing the standard Wind Sumo minigame.
